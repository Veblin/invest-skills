## 九、依赖最小化原则

1. **akshare 优先**——一个库覆盖 A 股行情/财务/资金流/情绪，减少额外依赖
2. **FRED + Tavily 两条主线**——宏观和搜索各自只用一条权威免费源即可满足 MVP
3. **SearXNG 做终极兜底**——避免依赖任何需要 API Key 的搜索服务
4. **Tushare 不纳入主 fallback 链**——用户主动配置 Token 后才启用，文档说明其 120 积分的实际限制
5. **每个依赖都追问"能否被 akshare/FRED/WebSearch 替代"**
