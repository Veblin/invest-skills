# 港股特殊性知识库

## 港股市场的独特性

港股是一个离岸市场，资金来自全球，标的以内地公司为主。这就是"中国的基本面、全球的资金面"。

## 做空机制

### 港股做空

港股允许做空（卖空），并且有**公开的做空数据**：

- **做空比率**：当日做空成交额 / 总成交额
- **做空比率 > 20%** → 需要关注

### 为什么重要

1. 做空的机构通常做了更多研究
2. 高做空比率可能是"先发现问题的聪明钱"
3. 也可能是做空资金过度的"轧空"机会

### 轧空

当大量做空仓位遇到股价上涨时，做空方被迫买回平仓 → 进一步推高股价 → 更多做空方被迫平仓 → **轧空**。

## 港元联系汇率制度

### 制度原理

港元与美元挂钩（1 USD ≈ 7.75–7.85 HKD），金管局通过外汇基金维持汇率。

### 对投资的影响

1. **港元利率跟随美元利率** → 美联储加息 = 港股资金成本上升
2. **汇率风险**：大陆投资者买港股需换汇 → 人民币升值 = 港股投资以人民币计收益下降
3. **联系汇率制度在极端情况下可能被打破**（极小概率但需知晓）

## 南向资金

### 什么是南向资金

南向资金 = 通过港股通从内地流入香港的资金。

### 关注点

- 南向资金偏好：腾讯、美团、中海油等大盘股
- 南向持续流入 → 港股流动性改善
- 南向大幅流出 → 内地资金避险信号

### 港股通标的限制

不是所有港股都能通过港股通购买。港股通有标的名单，定期调整。

## AH 溢价

### 什么是 AH 溢价

A+H 两地上市的公司，A 股和 H 股的价格通常不一样。

AH 溢价指数 = A 股价格 / H 股价格（汇率调整后）。

- **AH 溢价 > 1**（常态，A 股贵过 H 股）
- **AH 溢价 < 1**（倒挂，H 股贵过 A 股，较罕见）

### 为什么 A 股通常比 H 股贵

1. A 股有打新制度的额外价值
2. A 股流动性更好
3. A 股投资者结构（散户多，估值容忍度高）
4. 资本管制（人民币不能自由兑换）

### 投资启示

- AH 溢价处于历史高位 → 港股相对便宜
- AH 溢价处于历史低位 → A 股相对便宜
- 不是所有溢价都会"收敛"——制度性因素长期存在

## 港股交易规则

| 特征 | 港股 | A 股 |
|------|------|------|
| T+0 | 是 | 否（T+1） |
| 涨跌停 | 无 | ±10%/±20% |
| 最小交易单位 | 手（每手股数由公司自定） | 100 股 |
| 交易货币 | 港币 | 人民币 |
| 印花税 | 0.13% | 0.05% |

## 学习要点

1. **做空比率是港股的"市场情绪温度计"**，A 股没有这个指标
2. **联系汇率制度意味着港股的利率环境由美联储决定**
3. **南向资金是内地投资者对港股的投票**
4. **AH 溢价是两地市场估值差异的量化指标**
5. **港股波动大**：无涨跌停 + 全球资金快进快出 → 单日涨跌 10-20% 并不罕见
