# 全球宏观分析框架知识库

## 为什么 A 股投资者需要关注全球宏观

A 股虽是境内市场，但全球宏观变量通过以下渠道影响 A 股：

1. **资金流动**：美元走势 → 全球资金流向 → 北向资金
2. **估值锚**：美债收益率 → 全球资产估值的"地板"
3. **风险偏好**：VIX → 全球资金风险偏好 → 外资流入/流出新兴市场
4. **汇率传导**：人民币兑美元 → 出口竞争力、外汇储备

## 核心传导链

### 1. 美债收益率 → 全球资产估值

```
10年期美债收益率（全球无风险利率的锚）
    → DCF 模型中 WACC 上升
    → 远期现金流的现值下降
    → 高估值成长股（远期现金流占比大）受冲击最大
    → 低估值价值股（近期现金流占比大）受影响较小
```

**关键观察**：
- 10Y 美债收益率上升 → 压缩 PE 倍数 → 成长股估值压力
- 10Y 美债收益率下降 → PE 扩张空间 → 利好成长股

### 2. 美元指数（DXY）→ 资金流动

```
美元走强
    → 资金回流美国
    → 新兴市场资金外流
    → A股/港股承压
    → 人民币贬值压力

美元走弱
    → 资金流入新兴市场
    → 北向资金增加
    → A股/港股受益
```

**关注**：美元贸易加权指数（FRED: DTWEXBGS），比 DXY 更全面。

### 3. 收益率曲线 → 经济预期

```
10Y-2Y 利差：
  正值（正常）→ 长期利率 > 短期利率 → 经济扩张预期
  倒挂（< 0） → 短期利率 > 长期利率 → 衰退预警

历史上每次收益率曲线倒挂后，12-18个月内出现衰退的概率 > 70%。
```

### 4. 实际利率 → 黄金/商品

```
实际利率 = 名义利率 - 通胀预期

实际利率上升 → 持有黄金的机会成本上升 → 金价下跌
实际利率下降 → 黄金吸引力上升 → 金价上涨
```

## CME FedWatch 利率预期

CME FedWatch 基于 30 天联邦基金期货价格计算 FOMC 会议加息/降息概率。

- 数据来源：WebFetch cmegroup.com/markets/interest-rates/cme-fedwatch-tool.html 或直接解析
- 注意：市场预期 ≠ 实际结果（市场经常犯错）

## 学习要点

1. **宏观分析不预测短期市场**，而是理解当前所处的大环境
2. **美债收益率是"全球资产定价锚"**，理解它才能理解估值变化
3. **美元周期和新兴市场资金流动高度相关**
4. **宏观数据有滞后性**（GDP 季报、CPI 月报），需要区分"已知"和"待公布"
5. **单一宏观指标意义不大，需要多指标交叉验证**
