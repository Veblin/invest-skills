# 财务指标解读知识库

## 核心原则

**财务分析的目的是理解"这家公司如何赚钱、赚的钱质量如何"，而不是做算术题。**

## ROE（净资产收益率）

ROE = 净利润 / 股东权益。

**巴菲特标准**：连续多年 ROE > 15% 是优质公司的特征。

### 杜邦分析

ROE 可以拆解为三个驱动因素：

ROE = 净利率 × 总资产周转率 × 权益乘数

| 驱动因素 | 含义 | 高质量特征 | 风险信号 |
|---------|------|-----------|---------|
| 净利率 | 产品定价权、成本控制能力 | 稳定或趋势向上 | 持续下滑 |
| 总资产周转率 | 运营效率 | 同行业比较 | 资产变重但收入不增 |
| 权益乘数 | 财务杠杆 | 适度（< 3x） | 高杠杆（> 5x）驱动的高 ROE |

**关键判断**：ROE 是来自高利润率（好）、高效率（好）、还是高杠杆（危险）？

- 茅台 ROE ~30%：净利率 ~52%，杠杆 ~1.3x → **高质量 ROE（利润率驱动）**
- 地产公司 ROE ~25%：净利率 ~8%，杠杆 ~5x → **杠杆驱动的高 ROE（风险高）**

## ROIC（投入资本回报率）

ROIC = NOPAT（税后净营业利润）/ 投入资本。

- **ROIC > WACC** → 公司在创造价值
- **ROIC < WACC** → 公司在毁灭价值

ROIC 排除了杠杆的影响，比 ROE 更能反映经营效率。

## FCF（自由现金流）

FCF = 经营性现金流净额 - 资本支出。

### FCF/净利润 比率

- **> 80%** → 利润含金量高，"真金白银"
- **50%–80%** → 正常范围
- **< 50%** → 需要关注：利润是否转化为现金？
- **< 0（长期）**→ 危险信号：净利润为正但经营现金流为负

### 经营CF/净利润

- **> 100%** → 利润质量好
- **< 100%** → 应收账款增加或存货积压

## 归母净利润 vs 扣非净利润

- **归母净利润**：归属于母公司股东的净利润
- **扣非净利润**：扣除非经常性损益后的净利润

**警惕**：归母远高于扣非 → 公司利润靠卖资产/政府补贴支撑，不可持续。

## 毛利率

毛利率 = （营业收入 - 营业成本）/ 营业收入。

- 高毛利率（> 60%）→ 强品牌、强定价权、弱竞争
- 毛利率持续下滑 → 警惕竞争加剧或成本压力

## 关键比率速查

| 检查项 | 健康区间 | 危险信号 |
|--------|---------|---------|
| 资产负债率 | < 50% | > 70% |
| 应收账款/营收 | < 10% | > 30%（回款差） |
| 存货周转天数 | 行业对比 | 趋势上升（积压） |
| 商誉/净资产 | < 10% | > 30%（并购暴雷风险） |
| 关联交易占比 | < 5% | > 20%（需查明） |

## 学习要点

1. **现金流是"照妖镜"**：利润可以粉饰，现金流很难造假
2. **ROE 的拆解比 ROE 本身更重要**：知道来源才知道可持续性
3. **横向比、纵向比、交叉比**：单一年份的单一指标意义不大
4. **警惕"纸面利润"**：营收增长但应收账款增速更快 → 可能是赊销冲量
5. **财报附注比三张表更重要**：关联交易、或有负债、会计政策变更都在附注里
