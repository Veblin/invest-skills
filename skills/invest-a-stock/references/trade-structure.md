# 交易结构分析

> 受 [SKILL.md](../SKILL.md) LAW 6a 约束。本节为假设一致性检验，非买卖建议。所有价格区间基于多情景估值推导。

## 核心原则

```
买入建议: "你应该在 95 元买入"              → 禁止（替人做决定）
入场区间: "中性情景锚定 95-120 元，盈亏比 1:1" → 允许（提供决策工具）
止损指令: "跌破 85 元止损"                 → 禁止（替人下指令）
假设失效: "悲观估值下限 80 元，跌破意味着市场定价比悲观更差" → 允许（描述含义）
```

**区分标准：是否替用户做决定。**

## 三个组件

### 1. 入场区间

基于多情景估值推导。每个区间必须：

| 要素 | 说明 |
|------|------|
| 情景锚定 | 对应乐观/中性/悲观哪个情景 |
| 价格区间 | 基于该情景的估值计算 |
| 对应的 Forward PE | 方便跨标的比较 |
| 假设前提 | 该区间成立需要什么条件 |
| 盈亏比 | `|上行空间| / |下行空间| × 概率比` |

**盈亏比计算：**
```
盈亏比 = (乐观上限 - 区间中值) × P(乐观) / (区间中值 - 悲观下限) × P(悲观)
       = 加权上行 / 加权下行
```

**多个区间的含义：**
- 悲观锚区间（盈亏比高）：安全边际大，但需要相信悲观情景不会更差
- 中性锚区间（盈亏比适中）：假设全部兑现的合理价，无安全边际
- 乐观溢价区（盈亏比<1）：乐观情景已定价，上行空间有限

### 2. 假设失效触发

基于 thesis 追踪，将每个核心假设映射为一个可观测的触发条件：

| 假设 | 触发器 | 触发后动作 |
|------|--------|----------|
| 毛利率企稳 | 连续两季 <33% | 下调至悲观情景，重新计算入场区间 |
| 字节 capex 高增 | 2027 capex 增速 <15% | 悲观概率上调至 50%+ |
| 利润持续增长 | PE 被动升 100x+ 且利润 miss >20% | 双杀确认，中性→悲观切换 |
| 应收可控 | 应收账款/净利润 >300% | 降低置信度，收紧入场条件 |

**触发 ≠ 离场。** 触发 = 启动重新评估流程。你重新评估后，由你决定。

### 3. 操作纪律

流程规则，不是交易规则：

| 规则类型 | 示例 |
|---------|------|
| 定期检查 | "季报后 48 小时内执行 thesis --update" |
| 事件驱动检查 | "宏观情景发生重大变化时，更新情景概率" |
| 区间纪律 | "价格进入悲观锚区间 ≠ 自动入场，需先验证悲观假设是否被证伪" |
| 假设完整性 | "若假设完整度 <50%，启动全面重评估" |
| 记录纪律 | "每次入场/离场写入 invest-a-journal，记录入场时的情景判断和假设状态" |

## 嵌入报告

完整报告在"关键观察节点"后、"主要风险"前添加"交易结构分析"节。详见 SKILL.md 简报模板。

### 模板

```markdown
## 交易结构分析

> ⚠️ 本节为假设一致性检验，非买卖建议。所有价格区间基于上文多情景估值推导，盈亏比 = |上行空间|/|下行空间| × 概率比。实际操作需结合个人资金管理与风险承受能力。

### 入场区间（基于情景锚定）

| 情景锚定 | 价格区间 | 对应 Forward PE | 假设前提 | 盈亏比 |
|---------|:------:|:-----:|---------|:-----:|
| 悲观情景锚 | XX~YY 元 | — | 悲观假设全部兑现 | >X:Y |
| 中性-悲观区间 | XX~YY 元 | — | 悲观叙事部分定价 | X:Y |
| 中性情景锚 | XX~YY 元 | — | 当前中性假设全部成立 | X:Y |
| 乐观溢价区 | XX~YY 元 | — | 乐观假设已定价 | <X:Y |

### 假设失效触发（离场条件）

| 条件 | 类型 | 对应假设 | 触发后动作 |
|------|------|---------|----------|
| 毛利率连续两季 <X% | 假设证伪 | 盈利质量 | 下调至悲观情景，重新计算 |
| [大客户] capex 增速 <X% | 核心叙事动摇 | 增长驱动 | 悲观概率上调至 50%+ |
| PE 被动升至 Xx+ 且利润 miss >X% | 双杀信号 | 估值+业绩 | 中性→悲观情景切换 |
| 应收账款/净利润 >X% | 财务质量恶化 | 现金流健康 | 降低置信度，收紧入场条件 |

### 操作纪律

1. **定期检查**：季报后 48 小时内执行 `thesis --update`，对照假设 vs 实际
2. **宏观驱动**：宏观情景（地缘/利率/NFP）发生重大变化时，更新情景概率
3. **区间纪律**：价格进入悲观锚区间 ≠ 自动入场，需先验证悲观假设是否已被证伪
4. **假设完整度**：持仓期间，若假设完整度 <50%（>半数假设被弱化/推翻），启动全面重评估
5. **记录纪律**：每次入场/离场写入 invest-a-journal，记录入场时的情景判断和假设状态
```
